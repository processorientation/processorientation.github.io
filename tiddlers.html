Customizable Header

There are two ways of header customization:

  1. Use tag $:/tags/SideBar to update header links. You can read How to add a new tab to the sidebar, the process is the same.
  2. Use tag $:/plugins/rmnvsl/krystal/header to update sections.

Ordering

You can use fields list-after and list-before to sort the items. See Order of Tagged Tiddlers for more information.

Currently, the styling of search bar is hardcoded to be right aligned, so everything listed after it will be in the right corner.

Installation

22nd August 2020 at 8:55pm

You can either:

  • save this whole ~TiddlyWiki by clicking the save button
  • add just the bare plugin by dragging the following link onto your project $:/plugins/rmnvsl/krystal
  • already installed? Check how to Update to the new version.

⚠ Warning

Although it's already possible to use this plugin (I do use it for my notes in the current development stage), it's an early alpha and has several drawbacks. Just to name a few:

  1. animations and transitions are buggy in some cases 🐛
  2. I'm still looking around how ~TiddlyWiki works 🤸‍♂️

Horizontal Story River

This plugin is an attempt to bring Andy Matuschak's Evergreen notes system into TiddlyWiki. I found it by reading articles by Anne-Laure Le Cunff.

Thanks both for the inspiration, my (thinking about) notes finally started to become (somehow) useful!

Made by Roman Veselý - @crazko, @rmnvsl

Features

  • horizontal story view for tiddlers (see Example 👉)
  • bi-directional links at the bottom of each tiddler
  • Customizable Header instead of sidebar
  • edit/view Toolbar option to maximize tiddler width
  • nice animations and transitions 🐮

Customizable in plugin settings

  • closing open tiddlers to the right
  • highlight links of open tiddlers
  • tiddler style (width & padding)

Recommended plugins

2nd October 2020 at 5:04am

Installing following plugins will make the work with your ~TiddlyWiki easier:

  • Relink - When renaming a tiddler, Relink can update the fields, filters, and widgets of all other tiddlers.
  • Edit-CompText - This plugin adds completion when editing the body of a tiddler.
  • TiddlyRemember - A tool that integrates the TiddlyWiki with the spaced-repetition-flashcards tool Anki
  • Streams - Streams is a keyboard friendly way to write and easily divide your text into separate tiddlers while writing.
  • Refnotes - Refnotes is a set of macros and stylesheets for creating abbreviations, footnotes and citations.

Toolbar option to maximize tiddler width

22nd August 2020 at 8:35pm

Visit $:/ControlPanel > Appearance > Toolbars > Edit Toolbar / View Toolbar.

Find maximize tiddler option, check the checkbox to make it visible in the toolbar and/or drag & drop to re-position.
